# Frontend Prototype (React + Vite)

Run locally:

```bash
cd frontend
npm install
npm run dev
```

API base URL defaults to `http://127.0.0.1:3000`.
You can override with:

```bash
VITE_API_BASE_URL=http://127.0.0.1:3001 npm run dev
```

Screens included:
- Projects
- Import Bid Package
- Package Dashboard
- Dealer Unlock
- Dealer Bid Entry
- Comparison Dashboard

This is a UI shell with mock data. Hook API calls to the Rails endpoints when backend is ready.
